# KANBAN_DEMO
Канбан — десктопное приложение для создания и печати производственных карточек. Обеспечивает полный цикл: ввод данных, визуализацию и печать в форматах A4 (4 карточки/лист) или A6. Поддерживает работу с SQL Server и демо-режим с локальными данными. Включает кэширование, логирование и защиту от повторных вызовов. Время отклика &lt; 1 секунды.
